Overview

The purpose of this assignment is to create a "thin slice" through a character animation pipeline completely from scratch. By the end of the assignment, students will be able to warp a mesh automatically into a pose simply by moving one point

Getting Started

Click here to download the starter code for this assignment. The entry point for running things is the file skin.html. You'll mainly be editing the files skin.js and skindebugplots.js

When you're finished, submit skin.js and skindebugplots.js

Part 1: FABRIK

For the first part, you'll implement the FABRIK algorithm that we discussed in class (more info at this link).
